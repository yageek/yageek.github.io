Logic Analyzer Client de sump.org sous Ubuntu

2010.12.03
Après avoir récemment acquis l'analyseur logique libe après de dangerous prototypes, j' ai voulu utiliser le client graphique sur ma distribution Ubuntu. Les binaires ne fonctionnaient pas du premier coups donc j'ai réaliser un petit script d'automatisation qui installe, télécharge et compile tout le nécessaire. Je me suis basé sur ant pour la compilation java.

Pour l'installation :

chmod 0777 install_l_la.sh
./install_l_la.sh

L'installateur bash se trouve à la page : http://yageek.free.fr/analyzer.
